# KANTEI - Steganography Forensics Tool for Images

A python Tool for Heuristic Detection of Images Steganography.

## Features

- LSB Analysis (chi2, imbalance)
- Simplified RS
- ELA for JPEG
- Metrics visualization
- LSB Injection for testing

## Usage

```bash
python stego-scanner.py imagen.png --plot --report out.json
```
